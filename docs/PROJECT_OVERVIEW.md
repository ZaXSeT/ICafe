# Project Overview: ICafe

## Project Vision
ICafe is a modern, startup-grade SaaS application designed to provide a premium digital experience for cafes. It allows customers to reserve tables, view real-time availability, and order food and drinks directly from the website.

## Goals
- Deliver a premium, Apple-level UI/UX.
- Provide a robust reservation and ordering system.
- Empower cafe owners and cashiers with efficient management dashboards.
- Ensure scalable architecture, clean code, and comprehensive documentation.

## Features
- **Customer Facing**: Landing page, interactive table reservation, digital menu, ordering & checkout, user profile, and history.
- **Cashier Dashboard**: POS-like interface, active reservations, order management, table occupancy.
- **Owner/Admin Dashboard**: Revenue analytics, menu management, employee management, reservation management, transaction reports.

## User Roles
- **Customer**: Browses menu, makes reservations, places orders.
- **Cashier**: Manages POS, confirms payments, updates order status, manages tables.
- **Admin/Owner**: Full access, views analytics, manages menus and employees.

## Tech Stack
- **Frontend**: Next.js 15 (App Router), TypeScript, TailwindCSS, shadcn/ui, Framer Motion, Zustand.
- **Backend**: Next.js Server Actions, Prisma ORM, PostgreSQL.
- **Authentication**: NextAuth/Auth.js (Role-based).
- **Deployment**: Vercel.

## Branding Direction
- **Aesthetic**: Modern, clean, premium, minimal, luxurious, and warm.
- **UI Elements**: Soft modern shadows, glassmorphism, smooth Framer Motion transitions, rounded corners, clean typography.

# UI/UX Guidelines

## Design System

### Colors
- **Primary**: A rich, warm coffee brown or deep espresso.
- **Secondary**: A soft cream or oat color for backgrounds.
- **Accent**: A vibrant color (e.g., matcha green or muted orange) for primary actions (buttons, highlights).
- **Dark Mode**: Deep charcoal backgrounds with soft, glowing accents.

### Typography
- **Headings**: Modern serif or clean geometric sans-serif (e.g., Playfair Display or Inter).
- **Body**: Highly readable sans-serif (e.g., Inter, Roboto).

### Components & Elements
- **Cards**: Soft, diffused drop shadows (glassmorphism in certain areas).
- **Corners**: Rounded (e.g., `rounded-2xl` or `rounded-3xl` for modern feel).
- **Buttons**: Smooth hover effects, slightly scaling up on hover.

### Animation Rules (Framer Motion)
- **Page Transitions**: Soft fade-in and slide-up.
- **List Items**: Staggered entrance animations.
- **Interactions**: Micro-interactions on buttons, inputs, and cards.

### General Feel
The UI must feel like a premium, high-end cafe experience—clean, uncluttered, and inviting.
